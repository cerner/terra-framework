# Terra Theme Context
A provider component for the terra theme context. This component is not intended for use outside of terra-application.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-theme-context`

## Usage

```jsx
import ThemeContextProvider from 'terra-theme-context/lib/ThemeContextProvider';
```
