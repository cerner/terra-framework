# Terra Date Input

The DateInput allows for easy data entry of known dates like birthdays, anniversaries, etc. The display of the DateInput is localized based on the locale but can be customized via the `displayFormat` prop. The DateInput uses the ISO 8601 format for date values (YYYY-MM-DD).

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-date-input`

## Testing

The Date Input uses `uuid` which changes the component's description id dynamically. To mock it in jest test teams can use `jest.mock`.
If teams are using `shallow` for the tests then the mock is not required. But if using `mount` then the teams must use the mock as given below.

```js
jest.mock('uuid/v4', () => () => '00000000-0000-0000-0000-000000000000');
```
