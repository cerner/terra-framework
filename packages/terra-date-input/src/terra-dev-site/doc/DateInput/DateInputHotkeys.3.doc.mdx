# Terra Date Input Hotkeys

| Hotkey | Description |
|-|-|
| T | Sets current date to today. |
| - | Decrements current date by 1 day, if blank or invalid, will set date to yesterday |
| + | Increments current date by 1 day, if blank or invalid, will set date to tomorrow |
