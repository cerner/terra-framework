# Table - Column Actions

### Description

This example demonstrates a [Table](/components/cerner-terra-framework-docs/table/about) that supports Column Actions.

#### Required Properties

The **[action](/components/cerner-terra-framework-docs/table/about#column) property** needs to be passed to a column with following properties:

* **label** property: The label for the action button. 
* **onClick** callback property: When the user clicks on an action button, this callback is requested.
