# Table Sections

### Description

This example demonstrates a [Table](/components/cerner-terra-framework-docs/table/about) with data grouped by sections.

### Properties required
* sections: The tablular data for the component.
* isCollapsible: This property determines whether or not a section is collapsible. Collapsible sections will contain a button within the section header.
* isCollapsed: When the section header of a collapsible section is clicked, this property is being toggled to control whether the section is expanded or collapsed.
