# Table Sections

### Description

This example demonstrates a [Table](/components/cerner-terra-framework-docs/table/about) with data grouped by sections.

### Properties required
* sections: When the user selects a column, this callback will be called with the columnId of the selected column.
* isCollapsible: When the user selects a column, this callback will be called with the columnId of the selected column.
* isCollapsed: When the user selects a column, this callback will be called with the columnId of the selected column.
