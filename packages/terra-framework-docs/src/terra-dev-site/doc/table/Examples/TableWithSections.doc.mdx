# Table Sections

### Description

This example demonstrates a [Table](/components/cerner-terra-framework-docs/table/about) component with information grouped by sections.

### Required Properties
* **sections**: The tablular information for the component.
* **isCollapsible**: This property determines whether a section is collapsible. Collapsible sections contain a button in the section header.
* **isCollapsed**: When users select the section header of a collapsible section, this property is toggled to control whether the section is expanded or collapsed.
