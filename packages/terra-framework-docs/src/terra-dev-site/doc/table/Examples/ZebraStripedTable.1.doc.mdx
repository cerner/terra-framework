# Zebra Striping

### Description
This example demonstrates a [Table](/components/cerner-terra-framework-docs/table/about) that has zebra striping enabled for its rows.

### Properties required
* **isStriped**: Boolean specifying whether or not the table should have zebra striping for rows. A value of true will enable zebra striping.  
Otherwise, there will be no zebra striping in the table component.
