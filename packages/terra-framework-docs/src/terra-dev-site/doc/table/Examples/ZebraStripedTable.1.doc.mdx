# Zebra Striping

### Description
This example demonstrates a [Table](/components/cerner-terra-framework-docs/table/about) component that has zebra striping enabled for its rows.

### Required Properties
**isStriped**: A Boolean value specifying whether the table has zebra striping for rows. A value of **true** enables zebra striping.
