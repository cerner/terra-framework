# Headerless Table

### Description
This example demonstrates a [Table](/components/cerner-terra-framework-docs/table/about) without column headers.
The column headers still exist in the DOM for proper accessibility support.  However, they are hidden from the user's view.
Since the column headers exist in the DOM and are visible to screen readers, this must be accounteed for during implementation.

### Properties required
* **hasVisibleColumnHeaders**: Boolean indicating whether or not the table columns should be displayed. Setting the value to false will hide the columns, but the voice reader will use the column header values for a11y.
A value of false will hide the column headers. Otherwise, the column headers will be visible.
