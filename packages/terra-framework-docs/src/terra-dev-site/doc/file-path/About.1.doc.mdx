# Terra FilePath

<Notice variant="caution" ariaLevel="2">
Terra File Path is currently under alpha release. There may be breaking changes between releases until it is stable with the 1.0 release.
</Notice>

The `terra-file-path` component displays the current location within a hierarchy. The last item is considered the current location and is not interactable.

Each location is rendered as a [`terra-hyperlink`](https://engineering.cerner.com/terra-ui/components/cerner-terra-core-docs/hyperlink/about).

The component is fully responsive -- links will be displayed face-up and flex based on the space available. Any items that can not fit in the available space will be rolled into an ellipsis menu.

Consumers are responsible for maintaining the state of the component.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-file-path`
