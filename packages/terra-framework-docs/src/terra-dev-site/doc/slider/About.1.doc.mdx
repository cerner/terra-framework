## Overview

Terra Slider provides a way to select a value from a given range by sliding a thumb along a bar to track the value.

<Notice variant="caution" ariaLevel="2">
Terra Slider is an input component used to quickly select an imprecise value within a range such as a zoom level. It does not yet include features required for entering precise values.
</Notice>

<DefaultSlider />

### Features
Supports:
* Management of custom slider label, minimum label and maximum label.
* Setting default value for slider.
* Disabling of slider.
* Making other updates based on slider value change using the 'onChange' callback prop.
* Externally updating the slider value using the 'value' prop.

Refer to the [examples](/components/cerner-terra-framework-docs/slider/about#examples) section for detailed demonstration of the features supported by the Search Field.

## Accessibility
### Keyboard Interactions

List of keyboard interactions supported by Slider

|Key interactions|Description|
|---|---|
|Up Arrow    | Increases the value of the slider by one step.|
|Down Arrow     | Decreases the value of the slider by one step. |
|Right Arrow    | Increases the value of the slider by one step.|
|Left Arrow     | Decrease the value of the slider by one step.|
|Page Up    | Increases the value of the slider by ten steps.|
|Page Down     | Decrease the value of the slider by ten steps.|
|Home           | Set the slider to the first allowed value in its range.|
|End            | Set the slider to the last allowed value in its range.|
|Tab           | Moves focus into the slider on the thumb. If focus is already on the slider, it moves focus to the next focusable element in the tab sequence.|
|Shift + Tab    | Moves focus to the previous focusable element in the tab sequence.|

### Assistive Technology Support
#### WAI ARIA Roles and States
* [aria-label](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-label)
  * aria-label value is set to the 'label' prop value.
* [aria--describedby] (https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-describedby)
  * aria-describedby value is set to a visually hidden text with instructions to adjust the slider value between minimum and maximum values.
