# Calendar Filter

CalendarFilter is an sub-component of [terra-date-picker](https://github.com/cerner/terra-framework/tree/master/packages/terra-date-picker) component which renders faceup calendar for users to select a date from the calendar filter.

CalendarFilter will be useful in scenarios like when we need a simple calendar to be displayed instead of dropdown version of calendar, when date selection should happen within specfic range of date ( ex: user can make date selection in current month ) and etc.

## Getting Started

- Install with [npmjs](https://www.npmjs.com):
  - `npm install terra-date-picker`

## Implementation Notes:
CalendarFilter is required to be composed inside the [Base](https://engineering.cerner.com/terra-core/components/terra-base/base/base) component with locale in order for it to load the correct date format and translation strings.

## Usage
```js
import CalendarFilter from 'terra-date-picker/lib/CalendarFilter';
```
